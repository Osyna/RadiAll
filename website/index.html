FEATURES

What the wheel actually does

Nine things, and none of them ask you to remember a config-file syntax.

Launch or jump

Super + A fans your pinned apps out under the cursor. Click one to launch it, or land on it directly if it's already open.

Switch windows

Super + W shows every open window, grouped by app. If one app has three windows, scroll it and flip between them.

See it before you land

The middle of the ring shows a live thumbnail of the window you're pointing at. Scroll a busy app and each window previews in turn.

Act on the focused window

Super + D wraps the current window in close, float and fullscreen, plus the app's own .desktop actions like New Window.

Fire your own shortcut

Add a wedge that sends a keystroke into the live window — a New Tab button that presses Ctrl+T on whatever's focused. You set the keys once.

Real app icons

It reads each app's actual icon from your desktop theme — nothing to set up. For custom actions, search a library of thousands, or point at any image file.

Themes in one file

Switch from a dropdown, or drop a .json in the themes folder and it turns up there. Accent and background get a real colour picker. Ships with RadiAll, Paper, Light, Catppuccin and Nord.

One command to bind

A radiall command opens any ring, so a Hyprland bind is one line: radiall --apps. Run radiall --binds to print them all.

Tuned from a panel

Icon size, ring size, backdrop dim, wheel opacity, the shortcuts — all live in a settings panel. Hand-editing is there if you want it, not required.
